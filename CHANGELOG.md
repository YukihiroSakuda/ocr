# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-03

### Added
- Multi-page PDF support with page navigation controls
- Drag & drop file upload functionality
- Visual drag overlay feedback
- Custom application icon (neon cyberpunk clipboard design)

### Changed
- **Major UI redesign**: Sharp tech-focused cyberpunk aesthetic
  - Neon cyan (#00ffff) and pink (#ff0080) color scheme
  - Pure black background with subtle gradients
  - Monospace typography for technical elements
  - Sharp edges and geometric patterns
- Favicon updated to match application icon
- Improved upload zones with detailed technical descriptions

### Fixed
- DevTools now only opens in development mode (not in production)
- PDF text rendering for CJK fonts with CMap configuration
- History limit enforcement

### Removed
- Theme switching functionality (locked to dark mode only)
- Debug console.log statements from production build

## [0.2.0] - Previous releases

Initial development releases with core OCR functionality, history management, and settings.
